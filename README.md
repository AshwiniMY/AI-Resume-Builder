# AI Resume Builder
> A professional AI-powered tool to build ATS-friendly resumes in minutes.

[Live Demo](https://ai-resume-builder-4qga74eh3-ashwinimyaraguppi-4846s-projects.vercel.app/) • [Screenshots](#screenshots)

## 🛠 Tech Stack
React · Tailwind CSS · Strapi CMS · PostgreSQL (NeonDB) · Google Gemini API · Clerk Auth

## ✨ Features
- **AI Content Generation:** Automatically generate professional summaries and bullet points using Google Gemini.
- **ATS Optimization:** Receive real-time ATS scores and actionable feedback to improve your resume quality.
- **Seamless Export:** Download your completed, formatted resume as a PDF with a single click.
- **Secure Authentication:** Robust user login and data protection powered by Clerk.

## 🚀 Getting Started
```bash
git clone https://github.com/AshwiniMY/AI-Resume-Builder.git
cd AI-Resume-Builder-main
npm install
cp .env.example .env
# Make sure to populate the .env file with your actual keys
npm run dev
```
