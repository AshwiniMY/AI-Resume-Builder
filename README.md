# AI Resume Builder
A professional AI-powered tool to build ATS-friendly resumes in minutes.

## Setup
```bash
git clone https://github.com/AshwiniMY/AI-Resume-Builder.git
cd AI-Resume-Builder
npm install
npm run dev
```

## What it does
- Automatically generates professional summaries and bullet points using Google Gemini.
- Provides real-time ATS scores and actionable feedback to improve your resume quality.
- Allows downloading your completed, formatted resume as a PDF with a single click.
- Offers secure user authentication and data protection powered by Clerk.

## Tech used
React, Tailwind CSS, Strapi CMS, PostgreSQL (NeonDB), Google Gemini API, Clerk Auth
