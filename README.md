# AI Resume Builder

A professional AI-powered tool to build ATS-friendly resumes in minutes.

## 🚀 Features
- **AI Content**: Generate summaries and bullet points with Google Gemini.
- **ATS Score**: Get real-time feedback on your resume quality.
- **Easy Export**: One-click PDF download.
- **Secure**: User authentication with Clerk.

## 🛠️ Tech Stack
- **Frontend**: React, Tailwind CSS
- **Backend**: Strapi CMS
- **AI**: Google Gemini API
- **Auth**: Clerk

## 💻 Setup
1. Clone the repo: `git clone https://github.com/AshwiniMY/AI-Resume-Builder.git`
2. Install dependencies: `npm install`
3. Run: `npm run dev`
